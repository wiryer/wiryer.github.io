5. EFS PVC

EFS 설치

[설치]

  1. EFS 생성(AWS Console)
  • Elatstic Files System 서비스 접속
    • Create EFS : VPC 선택 후 자동으로 AZ 선택.
    • 현재 생성된 EKS 내에서 접속하도록 EKS Subent 으로 수정: Pod 들이 실행되는 Secondary Subnet 으로 설정
    • Security Group 수정
      • **NFS Port(2049)**를 추가하고, EKS Security Group 을 Source로 선택해서, EKS pod들이 접근 할 수 있도록 한다.
  1. efs provisioner deployment 생성
  • EKS 에서 EFS 를 사용하기 위해서 중간에 provisioner를 만들어 EFS와 연동한다.

  1. storageclass 생성
